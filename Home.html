Project Description
A Centrafuse plugin for the Monkeyboard, a DAB, DAB+ and FM radio Board. http://www.monkeyboard.org/products/85-developmentboard/85-dab-dab-fm-digital-radio-development-board-pro

Please note that the APIs to control the board can only be used with the MonkeyBoard and its DLL and no other KeyStone implementation.

To compile the source a copy of the Centrafuse SDK is required, www.centrafuse.com


Its considered stable (final?), but please report any bugs you may have.


Please avoid forking this project if it can be avoided (and it usually can).
